3  APM Screens

3.7 Trends Screen

3.7.1 Section Overview

A trend is a graphical representation of changes to tag values over time. The trends screen allows the user to view plots of tag values as they occur. The user can choose for plotting from among a large collection of the more crucial tags. The user can view tag data in real-time as they occur, or he can scroll back in time to view historical data that was collected and saved days, weeks, and in some cases, months in the past. Plotting values of more than one tag together on the same chart may provide the user with insight into cause-and-effect relationships and is a valuable troubleshooting tool.

The Trends screen is accessed by clicking the (F7) Trends button in the region of the Main screen with the forest green background color. It can also be accessed from any screen by pressing the F7 function key on the APM console keyboard.

3.7.2 Details

The Trends screen appears as shown in Figure 3.69, where data related to control of the extruder's zone temperatures is being plotted. Much of this screen is customizable by the user. For example, the chart background color can be changed, as can the colors and line widths of the various tag traces. Traces can appear superimposed on the same chart, or each tag can be plotted on its own chart as they are stacked vertically. The operator can even select which tags he wants to see plotted. The topic of customization will be discussed later on .

  1. Tag Data Plots - Tag values are plotted as a series of data points connected by line segments. Each tag has a default line color and width, but these can be changed by the user.
  2. Tag Color and Range Legend - This legend (see detail in Figure 3.70) shows which trace colors are assigned to which tags, and provides the numerical range for each tag trace. In this example the ranges are the same for all tags in the group, but this is usually not the case.
  3. Tag Current Values - This legend (see detail in Figure 3.71) displays the current value of each of the tags being plotted.
  4. Horizontal Scroll Bar - The seven buttons on this scroll bar allow the user scroll backwards in time to view tag data that has been scrolled off the chart to the left. The middle of the seven scroll bar buttons allows the user to alternately pause and resume the plotting of current tag data. When plotting of current data is paused the user can scroll back and forth through past tag values. The button to the immediate left of the pause/continue button allows the user to go backward in time, five minutes for each button click, and display the traces for the tag data that were then collected and saved. The next button over on the left allows the user to scroll backward in time in ten minute increments. Clicking the left-most button will display traces for the oldest tag data retained by the APM computer; this data may be months old. The button to the immediate right of the pause/continue button allows the user to go forward in time in five minute increments. The next button over on the right allows the user to scroll forward in time in ten minute increments. Clicking the right-most button returns the user to the present time as current data will now be plotted.
  5. Date/Time - These two fields, one near the top of the screen and one immediately above the scroll bar buttons displays the date and time, respectively, for the data that is currently plotted. The date is displayed at the top and the time is displayed just above the scroll bar buttons.
  6. Tag Groups - All the tags for which trends can be viewed are arranged in functional groups (e.g., extruder-related tags, liquids-related tags, temperature related tags). These buttons allow the user to change the tag group currently enabled.

If the operator right clicks anywhere on the trend screen's black background, a menu appears with some useful options (see Figure 3.72). Repeatedly clicking the Scroll option causes the trend to screen to alternately pause and resume updating.

A very useful customization tool available to the operator is the Properties faceplate (see Figure 3.73), which can be brought forth by selecting Chart Properties menu option in Figure 3.72.

The operator will find that the options available on the Pens tab will be the ones that he most frequently uses. With this tab the operator can change the visibility of any of the traces on the plot, which is convenient with charts that are confusing because so many traces are displayed at once. The operator can also change trace colors and line weights and trace data ranges.

Other handy options are available on the Y-Axis tab, such as plotting all visible traces on separate charts, rather than all on the same chart. On the X-Axis tab the operator can specify a date and time from the past from which to retrieve and plot data historical data that has been captured in the past and stored on the APM computer's hard drive.
