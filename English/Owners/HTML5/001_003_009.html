3  APM Screens

3.9 Utility Screen

3.9.1 Section Overview

The Utility screen features a handful of buttons that allow the operator to execute some miscellaneous, mostly unrelated, functions that need to be performed occasionally. The Utility screen is accessed by clicking the “(F10) Utility” button in the region of the Main screen with the forest green background color. It can also be accessed from any screen by pressing the “F10” function key on the APM console keyboard.

3.9.2 Details

The Utility screen is reproduced in Figure 3.95.

  1. “Flush Extruder” Button - Clicking this button commands the PLC to execute the extruder flush utility. Refer to Extruder Barrel Flush Sequence for a detailed description of how this utility is used.
  2. “Stop Flushing Extruder” Button - Clicking this button commands the PLC to terminate the extruder flush utility. Refer to Extruder Barrel Flush Sequence for a detailed description of how this utility is used.
  3. “Normal Flush Cylinder” Button - Clicking this button commands the PLC to execute the preconditioner normal flush utility. Refer to Preconditioner Flush Sequences for a detailed description of how this utility is used.
  4. “Wet Flush Cylinder” Button - Clicking this button commands the PLC to execute the preconditioner wet flush utility. Refer to Preconditioner Flush Sequences for a detailed description of how this utility is used.
  5. “Stop Flushing Cylinder” Button - Clicking this button commands the PLC to terminate whichever preconditioner flush utility is being executed by the PLC. Refer to Preconditioner Flush Sequences for a detailed description of how this utility is used.
  6. “Activate Feed Test Mode” Button - Clicking this button commands the PLC to execute the dry feed test utility. Refer to Dry Feed Test Mode Sequence for a detailed description of how this utility is used.
  7. “Deactivate Feed Test Mode” Button - Clicking this button commands the PLC to terminate the dry feed test utility. Refer to Dry Feed Test Mode Sequence for a detailed description of how this utility is used.
  8. “Set Time & Date” button - Clicking this button instructs the PLC to update its current date and time to match that of the APM computer.
  9. “Restart Derived Tags” Button - Clicking this button causes the APM application to stop and restart updating APM derived tags. Sometimes, this may be needed to clear the “Lost Communications with PLC” alarm.
  10. “EXIT” Button - Clicking this button closes the Utility screen.
