4.11 Dry Feed Test Mode Sequence

The purpose of the PLC-managed dry feed test mode utility is to facilitate calibration of the loss-in-weight feed system. While this utility is being executed the PLC controls the surge bin slide gate, the live bin agitator, the screw feeder, the preconditioner, and the downspout.

To initiate an PLC-managed preconditioner flush, the operator brings up the Overview screen on the control room APM console and ensures that all extruder system devices are in Auto control mode. The operator then presses the F10 function key on the keyboard, and on the Utility dialog box that appears (see Figure 4.3 ), clicks the Activate Feed Test Mode button (1). There is no limit on how long the extruder system will remain in this mode. The operator can terminate the dry feed test mode at any time by clicking the Deactivate Feed Test Mode button (2).

  1. “Activate Feed Test Mode” Button - Clicking this button will command the PLC to initiate the dry feed test mode sequence.
  2. “Deactivate Feed Test Mode” Button - Clicking this button will command the PLC to terminate the dry feed test mode sequence.

The sequence of operations that the PLC executes when the operator chooses to enter the dry feed test mode are listed below in the order that they take place.

  1. The PLC causes the alarm horn to sound and the beacon to flash for about ten seconds to alert all personnel that the extruder system is about to start. At the same time the PLC displays the system status in the upper left corner of the APM monitor as “Dry Feed Test Mode.”
  2. The PLC fills the live bin with dry material from the Surge Bin. The Live Bin will automatically refill during normal operation each time the Live Bin weight falls below a predetermined value.
  3. The PLC starts the live bin agitator.
  4. The PLC starts the preconditioner in the forward direction.
  5. The PLC starts the screw feeder after it verifies that the preconditioner is running.
  6. Initially, the PLC calculates the feeder speed setpoint using the formula dry feed bulk density and the formula feed rate setpoint. However, soon after steady-state is achieved the PLC uses the Loss In Weight (LIW) algorithm to determine the feeder speed setpoint.

The sequence of operations that the PLC executes when the operator chooses to exit the dry feed test mode are listed below in the order that they take place.

  1. The PLC displays the system status in the upper left corner of the APM monitor as “Extruder Stopped.”
  2. The PLC stops filling the live bin with dry material from the Surge Bin.
  3. The PLC stops the live bin agitator.
  4. The PLC stops the preconditioner.
  5. The PLC stops the screw feeder.
