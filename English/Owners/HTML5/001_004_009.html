4.9 Extruder Barrel Flush Sequence

The extruder barrel flush sequence is a maintenance utility whose purpose is to assist is removing material from the extruder barrel by flushing it out with process water and steam. This utility sequence is intended to be performed only when the extruder system is not in operation, as after a production run is concluded, or just prior to the start of a production run but before the extruder die is attached. This sequence should be executed with all extruder system devices not running and with the extruder die removed.

To initiate an PLC-managed extruder barrel flush, the operator brings up the Overview screen on the control room APM console, presses the “F10” function key on the keyboard, and on the Utility dialog box that appears, see Figure 4.1, clicks the Flush Extruder button (1). The operator can prematurely terminate the extruder barrel flush by clicking the Stop Flushing Extruder button (2) (otherwise, the PLC by default terminates the sequence after 4 minutes)

  1. “Flush Extruder” Button - Clicking this button will command the PLC to initiate the extruder flush sequence
  2. “Stop Flushing Extruder” Button - Clicking this button will command the PLC to terminate the extruder flush sequence prematurely

The sequence of operations that the PLC executes when the operator chooses an extruder barrel flush are listed below in the order that they take place.

  1. The PLC causes the alarm horn to sound and the beacon to flash for about ten seconds to alert all personnel that the extruder system is about to start. At the same time the PLC displays the system status in the upper left corner of the APM monitor as “Flushing Extruder.”
  2. The PLC enables process water to flow to the extruder at an extruder flush rate configured by the Wenger Technicians during initial installation and system startup.
  3. The PLC enables process steam to flow to the extruder at an extruder flush rate configured by the Wenger Technicians during initial installation and system startup.
  4. The PLC starts the extruder's lubrication oil pump.
  5. The PLC starts the extruder motor at a speed that is 30 percent of the formula final extruder speed setpoint. The PLC will only start the extruder motor if it confirms the flow of lubrication oil. The PLC will terminate the normal startup sequence if lubrication oil flow is not confirmed within a time interval that was configured by the Wenger Technicians during initial installation and system startup. Otherwise the PLC will increase the extruder motor speed to a extruder flush speed that was configured by the Wenger Technicians during initial installation and system startup.
  6. The PLC allows flushing to continue for a maximum of four minutes from the point where the operator clicked the Flush Extruder button. During this time the operator can change the flow rate of process water or process steam to the extruder. To change the process water flow rate to the extruder, bring forth the extruder water PID dialog box, click the Manual button to change to Manual control mode, and change the percent output value. To change the process steam flow rate to the extruder, bring forth the extruder steam PID dialog box, click the Manual button to change to Manual control mode, and change the percent output value. The operator can terminate flushing prematurely at any time by clicking the Stop Flushing extruder button on the Utility dialog box.
  7. The PLC terminates flushing by first disabling flow of process water and process steam to the extruder. A few seconds later the PLC stops the extruder motor and extruder lube pump.
