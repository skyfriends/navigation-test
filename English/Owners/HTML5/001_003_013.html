3.13 Totalizer Screen

3.13.1 Section Overview

The Totalizer screen displays a running total of the mass of each product ingredient that have been consumed during production by the extruder system since the last time that these running totals have been reset to zero. The Totalizer screen is accessed by clicking the “(cT) Totalizer Info” button in the region of the Main screen with the gray background color.

3.13.2 Details

An example of the Totalizer screen, which is tailored to each customer's extruder system and their products' ingredients, is shown in Figure 3.103. In this example the customer uses ten liquids (besides the process steam and process water which are always used).

  1. “Reset” Button - Clicking this button copies the current ingredient totals from the Current column to the Previous column, and then resets all the current ingredient totals to zero.
  2. Ingredient Names Column - This column displays the names of the customer's various ingredients.
  3. Previous Column - Displays the totalized consumed amounts for each of the customer's ingredients just prior to when the “Reset” button was most recently clicked.
  4. Current Column - Displays the constantly-updated consumption total for each of the customer's ingredients amounts.
  5. Engineering Units Column - Displays the engineering units in which each totalized value is expressed.
  6. Consumed Dry Feed Totals
  7. Consumed Extruder Steam Totals
  8. Consumed Extruder Water Totals
  9. Consumed Preconditioner Steam Totals
  10. Consumed Preconditioner Water Totals
  11. Consumed Liquid 1 Totals - There will be as many rows for totalized liquids as the number of liquids that the customer uses to formulate his products.
