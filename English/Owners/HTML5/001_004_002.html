4  Programmable Logic Controller

4.2 Definitions

Knowing the following definitions will help make the procedures that follow easier to grasp.

Table 4.1 Definitions
PLC Programmable Logic Controller. The processor that interfaces with the extruder equipment on one side, and with the APM control station software on the other side. The software in the PLC is what directly controls the extruder system equipment.
HMI Human Machine Interface. Referred to also as the operator control station.
E-Stop Pushbutton Known also as the operator control station. Consists of APM computer, monitor, keyboard and mouse, alarm strobe and special purpose buttons, switches and lamps. These enable the operator to view operational parameters of, and control the extruder system.
