3.10 Maintenance Screen

3.10.1 Section Overview

Wenger recommends that customers perform certain reoccurring maintenance operations on their extruder systems to extend the operational life of the equipment and to minimize repair and replace costs. Such maintenance tasks include things like lubricating moving parts, cleaning filters, and changing oil. The Extruder Maintenance screen is intended to alert operators and maintenance personnel when extruder system scheduled maintenance tasks need to be completed, based on a running total of hours of extruder system operation. The Extruder Maintenance screen is accessed by clicking the “(F12) Maintenance” button in the region of the Main screen with the forest green background color. It can also be accessed from any screen by pressing the “F12” function key on the APM console keyboard.

3.10.2 Details

The PLC that controls the extruder system machinery maintains a running total of hours that the extruder system is in operation. The PLC uses this number to maintain a running total of the number of hours until each of certain routine scheduled maintenance tasks need to be performed. Some tasks can be performed by either operators or maintenance personnel, while other tasks are to be performed only by maintenance personnel.

An example of the Extruder Maintenance screen is reproduced in Figure 3.96. This is how the screen would look when the extruder system is started up at the customer's facility for the first time by the Wenger Service Team. It is also how it appears when a user has logged in who has a security level higher than the Operator user type.

  1. Extruder Hours - This is the running total of the number of hours that the extruder has been running since it was commissioned by the Wenger Service Team, 7501 hours in the example above. The PLC accumulates time only when the extruder motor is in the running state.
  2. Hours Until Maintenance Indicators - This column of numbers, each of which is maintained by the PLC and counts down only while the extruder motor is running, shows the total number of operational hours remaining until the corresponding scheduled maintenance task should be performed. In Figure 3.96, the task at the top of the list will need to be performed after four more hours of extruder operation.
  3. “Reset” Buttons - When it is time for a maintenance task to be performed the background color of the button associated with that task will change to red. Clicking any of the buttons in this column resets the corresponding hours-until-maintenance value to the left of the button to its maximum value.
  4. Maintenance Instructions - There are many small rectangular and circular maintenance instruction prompts displayed on the screen. When it is time for a maintenance task to be performed the background color of the prompt associated with that task will change to red. Hovering the mouse over any of these small rectangular/circular prompts causes the maintenance instructions associated with the associated equipment to be displayed, as in Figure 3.98.

An example of the Extruder Maintenance screen is reproduced in Figure 3.97 where it is time to perform a maintenance task.

  1. Hours until Maintenance Indicator - Notice that the countdown time for that maintenance task has decremented to zero (1), the background color of the reset button for that task has changed to red (2), and the maintenance instructions prompt for that task is colored red. (3)
  2. “Press Sensor” Button - Notice that the background color of the reset button for that task has changed to red (2), and the maintenance instructions prompt for that task is colored red. (3).
  3. Maintenance Instruction Prompt - Notice that the maintenance instructions prompt for that task is colored red. Hovering the mouse over the small red rectangular prompt causes the maintenance instructions associated with the associated equipment to be displayed, see Figure 3.98.

Users with Maintenance security levels can reset all maintenance task countdown timers, while users with Operator security levels can reset only a few. Figure 3.99 shows the reset buttons that are visible on the Extruder Maintenance screen when the user is logged in as a Maintenance user type. All buttons are visible. Figure 3.99 shows the reset buttons that are visible on the Extruder Maintenance screen when the user is logged in as an Operator user type. Only the top three buttons are visible.

  1. “Pressure Sensor” Button - The extruder's conehead pressure sensor needs to be greased.
  2. “Seals and Bearings” Button - The downspout seal and the knife and BPV bearings need to be greased.
  3. “Air Filters” Button - The downspout air filter needs to be cleaned.
  4. “Preconditioner/Feeder” Button - The feeder and preconditioner bearings need to be greased.
  5. “Knife Shaft” Button - The knife shaft needs to be lubricated.
  6. “Extruder Filter” Button - The extruder oil filter needs to be replaced.
  7. “Extruder Oil” Button - The oil in the extruder bearing housing needs to be changed.
  8. “Gearboxes” Button - The oil in the feeder gearbox needs to be changed.
  9. “Motor Bearings” Button - The extruder motor bearings need to be greased.
