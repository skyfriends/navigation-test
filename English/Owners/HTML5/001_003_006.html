3.6 Extruder Liquids 2 Screen

3.6.1 Section Overview

Although the Extruder Liquids 2 screen is not a standard feature for all extruder systems, it is discussed here since most customers have product formulations that require at least one liquid other that water and steam. Examples of such liquids are dyes and oils. This screen also displays the most important formula-specific operating conditions on the right-side of the screen from which the corresponding setpoints can also be changed.

If an Extruder Liquids 2 screen is provided, it is not accessible from the Main screen, but rather by clicking an appropriately-labeled button at the bottom of the Extruder Liquids 1 screen.

3.6.2 Details

An example of the Extruder Liquids 2 screen is reproduced in Figure 3.64. This screen shows a PID control loop faceplate for five customer-specific liquids, ranging from liquid 1 on the left end to liquid 5 on the right end. For each liquid, there is a PID control loop faceplate, but also a graphic below the faceplate with which the operator can monitor and control other liquid features. The aspects of this screen pertaining to liquid 3 as an example and will be discussed below.

  1. Liquid 3 PID Control Loop Faceplate - This PID control loop faceplate for liquid 3, like the other four liquid faceplate’s, provides the most status and control options concerning control loop behavior.
  2. Liquid 3 Agitator Status/Control - This indicator and button allow the operator to monitor and control the liquid 3 tank agitator behavior. The agitator motor graphic color indicates the run status of the agitator, where green indicates that the agitator is running, while blue indicates that the agitator is not running. The text in the ellipse to the immediate right of the motor graphic, which displays either “On” or “Off”, indicates the same thing. The color inside the ellipse indicates the agitator's HOA control mode, white for Auto control, yellow for Hand (i.e., Manual) control, and blue for Off.
  3. Liquid 3 Tank Refill Status/Control - This indicator and button allow the operator to monitor and control the liquid 3 tank refill behavior. The text in the ellipse, which displays either “Refill” or “Off”, indicates whether the PLC is requesting that the tank be refilled (the customer is responsible for monitoring this signal and refilling the tank when requested). The color inside the ellipse indicates the agitator's HOA control status, white for Auto control, yellow for Hand (i.e., Manual) control, and blue for Off.
  4. Liquid 3 Pump Status/Control - This indicator and button allow the operator to monitor and control the liquid 3 tank pump behavior. The pump motor graphic color indicates the run status of the pump, where green indicates that the pump is running, while blue indicates that the pump is not running. The text in the ellipse to the immediate right of the motor graphic, which displays either “On” or “Off”, indicates the same thing. The color inside the ellipse indicates the pump's HOA control mode, white for Auto control, yellow for Hand (i.e., Manual) control, and blue for Off.
  5. Liquid 3 High Level Sensor Indicator - There are two liquid level indicators on the liquid 3 tank, one for indicating high level and the other below for indicating low level. For each indicator, a green color indicates that the liquid level in the tank is above the sensor, while a white color indicates that the liquid level in tank is below the sensor.
  6. Setpoints and Actual Values of Most Important Operating Conditions - This panel displays the setpoints and current values of the operating conditions that are most crucial to the current product's formulation.
  7. Navigating Buttons - Some screens, such as this one, provide buttons at the bottom of the screen that allow the user to quickly switch to other screens without needing to go to the Main screen first.

The top-right corner of the Extruder Liquids 2 screen has been reproduced in Figure 3.65. The liquid flow rates are displayed in the right-most column. To the left is a column that displays, in white background colors, the liquid flow rate setpoints as percentages of the dry feed rate setpoint. To change a liquid's flow rate setpoint, click on the corresponding setpoint in a white rectangle. A faceplate will appear from which you can enter a new setpoint as a percentage of feed rate.

  1. Liquid 3 Setpoint and Flow Rate

For any of the five liquids inFigure 3.65, clicking the button with the agitator motor graphic causes an HOA faceplate to appear, from which the operator can change the agitator's control mode. Since agitators are On or Off devices (they have fixed-speed motors), the operator's only available control options are selecting the agitator's control mode; there are no setpoint or speed control parameters. The liquid 3 agitator HOA faceplate in Figure 3.66, side [A], center [B], side [C], shows the three possible states for the liquid 3 agitator. In Figure 3.66, side [A] the agitator has been placed in Auto mode and is running. In Figure 3.66, center [B] the agitator has been placed in Hand (Manual) mode and is running. In Figure 3.66, side [C] the agitator has been placed in Off mode and is stooped.

  1. Device Name Indicator - “Liquid 3" in Figure 3.66, side [A], center [B], side [C].
  2. State Indicator - “Running” in Figure 3.66, side [A] and Figure 3.66, center [B], and “Stopped” in Figure 3.66, side [C].
  3. Mode Indicator - “Auto” in Figure 3.66, side [A] , “Hand” in Figure 3.66, center [B], and “Off” inFigure 3.66, side [C].
  4. “Hand” Button - Clicking this button will instruct the PLC to place the device in the Hand control mode.
  5. “Off” Button - Clicking this button will instruct the PLC to place the device in the Off control mode.
  6. “Auto” Button - Clicking this button will instruct the PLC to place the device in the Auto control mode.
  7. “Exit” Button - Closes the HOA faceplate.

For any of the five liquids inFigure 3.64, clicking the button inside the tank causes an HOA faceplate to appear, from which the operator can change the tank's refill control mode. Since refill are either On or Off the operator's only available control options are selecting the agitator's control mode; there are no setpoint or speed control parameters. The tank refill HOA faceplate is similar enough to the agitator HOA faceplate in Figure 3.66, side [A], center [B], side [C] that they won't be discussed in detail.

The PID control loop faceplate’s for the four liquids have identical functionalities. Each faceplate allows the operator to place the corresponding flow rate PID control loop in either Auto control mode or Manual control mode. In Auto mode, the PLC varies the respective pump speed to maintain a constant flow rate that is determined from the setpoint. In Manual control mode, the PLC maintains the pump speed as per the operator's input. In Figure 3.67, side [A], the liquid 3 flow rate control loop is in Auto control mode, while in Figure 3.67 , side [B], the liquid 3 flow rate control loop is in Manual control mode. The PLC ensures that the flow rate remains constant when the operator changes between Auto control mode and Manual control mode; this is referred to as “bumpless” control mode switching.

  1. “Auto Control Mode Selector” Button - Clicking this button will cause the PLC to switch from Manual control mode to Auto control mode.
  2. “Manual Control Mode Selector” Button - Clicking this button will cause the PLC to switch from Auto control mode to Manual control mode.
  3. Control Mode Indicator - Indicates the control loop's current control mode, Auto in the Figure 3.67, side [A] and Manual in Figure 3.67, side [B].
  4. Auto Control Mode Flow Rate Setpoint Select - This group of controls allow the operator to change the current flow control loop's setpoint, ramping up or down by clicking the arrow buttons, or entering a numerical value by clicking inside the white rectangular field. Notice in Figure 3.67, side [B] that the setpoint is only displayed but cannot be changed when in Manual control mode.
  5. Device Name - “Liquid 3" in Figure 3.67. This group of controls allow the operator to change the pump's speed output command, ramping up or down by clicking the arrow buttons, or entering a numerical value by clicking inside the white rectangular field. Notice in Figure 3.67, side [A] that the pump speed output command is only displayed but cannot be changed when in Auto control mode.
  6. Setpoint Indicator - Bar graph that displays the current flow rate setpoint in units of weight/hour (converted from percent of dry feed rate).
  7. Flow Rate Indicator - Bar graph that displays the current flow rate actual value in units of weight/hour (converted from percent of dry feed rate). The color of this bar graph indicates whether the flow rate is within safe operating range (blue) or outside safe operating range (yellow).
  8. Safe Operating Range Indicator - This bar graph displays the safe operating range for the flow rate (blue).

For any of the five liquids in Figure 3.64, clicking the button with the pump motor graphic causes a pump HOA faceplate to appear that is very similar to the liquid flow rate PID control loop faceplate. In fact, the liquid pump HOA faceplate provides the same functionality as the liquid flow rate PID control loop faceplate, but additionally allows the operator the ability to determine the pump's On/Off status. In Figure 3.68, side [A], center [B], side [C], only the On/Off status of the liquid 3 pump is being changed; the status of the liquid 3 flow PID control loop is not affected.

  1. “Auto” Button - Clicking this button will instruct the PLC to decide when the pump should be On or Off, but the speed of the pump is determined by the liquid 3 flow PID control loop.
  2. Mode Indicator - The liquid 3 pump is in Auto control mode after the Auto button was clicked.
  3. State Indicator - The liquid 3 pump is running as long as the PLC is satisfied that all necessary conditions for it to run in Auto mode are satisfied.
  4. “Hand” Button - Clicking this button will instruct the PLC that the operator wants the pump to run, overriding the o decide when the pump should be On or Off, but notice that the speed of the pump is still determined by the liquid 3 flow PID control loop.
  5. Mode Indicator - The liquid 3 pump is in Hand control mode after the Hand button was clicked.
  6. State Indicator - The liquid 3 pump is running as long as the PLC is satisfied that all necessary conditions for it to run in Hand mode are satisfied.
  7. “Off” Button - Clicking this button will instruct the PLC to place the device in the Off control mode.
  8. Mode Indicator - The liquid 3 pump is in Off control mode after the Off button was clicked.
  9. State Indicator - The liquid 3 pump is stopped.
  10. Flow PID Control Loop Output - Since the pump is not running there is no liquid 3 flow. But the liquid 3 flow PID control loop, which is still in Auto control mode and does not know whether the pump is running, tries to increase the pump speed to increase the liquid 3 flow rate. It eventually increases the pump speed command to its maximum 100%.
