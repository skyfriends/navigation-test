4.10 Preconditioner Flush Sequences

There are two preconditioner flush sequences whose purpose is to assist is removing material from the preconditioner: the normal flush sequence and the wet flush sequence. The only difference between these two sequences is that the normal flush does not involve injecting process water or process steam into the preconditioner while the wet flush does. The normal preconditioner flush sequence is intended to be executed first, followed by the wet preconditioner flush, the rationale being that it is preferable to remove as much of the dryer material in the preconditioner before attempting to flush out the remainder with water. This utility sequence is intended to be performed only when the extruder system is not in operation, as after a production run is concluded, or just prior to the start of a production run. This sequence should be executed with all extruder system devices not running and with the downspout placed in the Bypass position.

To initiate an PLC-managed preconditioner flush, the operator brings up the Overview screen on the control room APM console, presses the F10 function key on the keyboard, and on the Utility dialog box that appears (see Figure 4.2 ), clicks the Normal Flush Cylinder button (1). The operator can prematurely terminate the flush sequence by clicking the Stop Flushing Cylinder button (3) (otherwise, the PLC by default terminates the sequence after 4 minutes). Following this, the operator clicks the Wet Flush Cylinder button (2). Again, the operator can prematurely terminate the flush sequence by clicking the Stop Flushing Cylinder button (3) (the PLC otherwise terminates the sequence after 4 minutes).

  1. “Normal Flush Cylinder” Button - Clicking this button will command the PLC to initiate the cylinder normal flush sequence.
  2. “Wet Flush Cylinder” Button - Clicking this button will command the PLC to initiate the cylinder wet flush sequence.
  3. “Wet Flush Cylinder” Button - Clicking this button will command the PLC to terminate whichever cylinder flushing sequence is in progress.
