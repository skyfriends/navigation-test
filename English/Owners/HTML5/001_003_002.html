3  APM Screens

3.2 Main Screen

3.2.1 Section Overview

This chapter will discuss the four rectangular color-coded regions of the Main screen and the various buttons on the Main screen. The Main screen is the single top-level screen in the APM hierarchy of screen. This is the screen that first appears after the APM computer boots up and begins executing the APM software application. It provides the user with access to all second-tier screens. The user can quickly return to the Main screen by pressing the Home key on the computer keyboard or by clicking the “(Home) Main” button in the screen header with the mouse.

3.2.2 Screen Regions

The Main screen is divided into four regions (see Figure 3.27). Each region is colored to indicate that all the buttons residing within each respective region share a common functionality at a higher level. The colors of the four colored regions of the Main screen are:

  • “Forest Green” Color - indicating that all buttons in this region pertain to operation of the extruder system.
  • “Dark Yellow” Color - indicating that all buttons in this region pertain to operation of a dryer system.
  • “Lavender Blue” Color - indicating that all buttons in this region pertain to operation of a cooler/coater system.
  • “Gray” Color - indicating that all buttons in this region perform general purpose functions.
  1. Extruder System Buttons - The buttons in this region with the forest green background color allow the user to call up screens that are generally related to managing the operation of the extruder system. Some extruder system features are optional, and most customers will not purchase an extruder systems with all the optional features. Seemingly random empty spaces in the extruder system region are where buttons related to optional features not purchased are hidden.
  2. Dryer System Functions - The buttons in this region with the dark yellow background color allow the user to call up screens that are generally related to managing the operation of the dryer system. If a customer purchases a Wenger extruder system only, without an accompanying dryer system, this region will be empty or very nearly empty of buttons. In addition, some dryer system features are optional, in which case seemingly random empty spaces in the region are where buttons related to optional features not purchased are hidden. In Figure 3.27, it appears that the customer may have purchased a dryer from a different vendor and only chose to buy dryer-related equipment from Wenger that allows him to measure product bulk density and moisture.
  3. Cooler/Coater System Functions - The buttons in this region with the lavender blue background color allow the user to call up screens that are generally related to managing the operation of the cooler/coater system. In Figure 3.27, it appears that the customer did not purchase a cooler/coater system.
  4. General Purpose Functions - The buttons in this region with the gray background color allow the user to call up screens that are general in nature (not specific to the extruder, dryer, or cooler/coater) or they are functions that are not available at the operator level. Buttons in this region that are visible but disabled (do not respond to mouse clicks) are accessible only to users who log in using an APM software user account with higher privileges than what is allowed with the “Operator” user account.

3.2.3 Extruder System Function Buttons

The function buttons that pertain only to the extruder system reside in the region of the Main screen with the “Forest Green” background color. (see Figure 3.28). These buttons cause lower level screens or faceplate's to be displayed as follows:

  1. “(F1) Extruder Overview” Button - Clicking this button (or alternately pressing the F1 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Overview screen. This screen is discussed in detail in See Section 3.3, Extruder Overview Screen..
  2. “(F3) Barrel Screen” Button - Clicking this button (or alternately pressing the F3 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Barrel Temperatures screen. This screen is discussed in detail in See Section 3.4, Extruder Barrel Temperatures Screen..
  3. “(F4) Liquid Control” Button - Clicking this button (or alternately pressing the F4 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Liquids 1 screen. This screen is discussed in detail in See Section 3.5, Extruder Liquids 1 Screen..
  4. “(F5) Extruder Command” Button - Clicking this button (or alternately pressing the F5 function key on the keyboard from any APM screen) brings forth Commands Menu faceplate and places it on top of the currently displayed full-size screen. This faceplate is discussed in detail in See Section 3.3, Extruder Overview Screen..
  5. “(F6) Bypass/Confirm Menu” Button - Clicking this button (or alternately pressing the F6 function key on the keyboard from any APM screen) brings forth the Bypass/Confirm menu faceplate and places it on top of the currently displayed full-size screen. This faceplate is discussed in detail in See Section 3.3, Extruder Overview Screen..
  6. “(F7) Trends” Button - Clicking this button (or alternately pressing the F7 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Trends screen. This screen is discussed in detail in See Section 3.7, Trends Screen..
  7. “(F8) Formula” Button - Clicking this button (or alternately pressing the F8 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Formula screen. This screen is discussed in detail in See Section 3.8, Formula Screen..
  8. “(F10) Utility” Button - Clicking this button (or alternately pressing the F10 function key on the keyboard from any APM screen) brings forth the Utility menu faceplate and places it on top of the currently displayed full-size screen. This screen is discussed in detail in See Section 3.9, Utility Screen..
  9. “(F12) Maintenance” Button - Clicking this button (or alternately pressing the F12 function key on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Extruder Maintenance screen. This screen is discussed in detail in See Section 3.10, Maintenance Screen..
  10. “Extruder Bulk Density” Button - Clicking this button causes the full-size Main screen to be replaced with the full-size Extruder Bulk Density screen. The bulk density system is not considered a standard extruder system feature, and so it won't be discussed in detail in this document.

3.2.4 General Purpose Function Buttons

The function buttons that do not pertain specifically to either the extruder, dryer or cooler/coater systems reside in the region of the Main screen with the “Gray” background color. (see Figure 3.29). These buttons cause lower level screens or faceplate's to be displayed as follows:

  1. “Extruder LIW Configuration” Button - This button is disabled and does not respond to clicks unless the user is logged on as a “Wenger” user type. For Wenger user types, clicking this button causes the full-size Main screen to be replaced with the full-size Extruder Loss In Weight (LIW) Configuration screen. Since this screen is not accessible to the customer's operators this screen will not be addressed further in this document.
  2. “Extruder Configuration” Button - This button is disabled and does not respond to clicks unless the user is logged on as a “Wenger” user type. For Wenger user types, clicking this button causes the full-size Main screen to be replaced with the first of several full-size Extruder Configuration screens. Since these screens are not accessible to the customer's operators these screens will not be addressed further in this document.
  3. “(F9) Alarm Summary” Button - Clicking this button (or alternately pressing the F9 function key on the keyboard from any APM screen) brings forth the Alarm Summary menu faceplate and places it on top of the currently displayed full-size screen. This screen is discussed in detail in See Section 3.11, Alarm Summary Screen..
  4. “Alarm Log Viewer” Button - Clicking this button causes the full-size Main screen to be replaced with the full-size Alarm Log Viewer screen. This screen is discussed in detail in See Section 3.12, Alarm Log Viewer Screen..
  5. “(cT) Totalizer Info” Button - Clicking this button (or alternately pressing the ctrl+'T' key combination on the keyboard from any APM screen) causes the full-size Main screen to be replaced with the full-size Totalizer screen. This screen is discussed in detail in See Section 3.13, Totalizer Screen..
  6. “Stop Project” Button - Clicking this button terminates the APM application. There are no other methods of terminating the application as are available with other Windows applications.
  7. “(csL) Log On” Button - Clicking this button, or alternately pressing the control+shift+'L' key combination from any APM screen, brings forth the Client Login faceplate, shown in Figure 3.30, and places it on top of the currently displayed full-size screen. By using this faceplate the user can change the user type he is currently logged in as.
  8. “Print Data Sheet” Button - Clicking this button brings forth the faceplate that summarizes the operating conditions currently being used to make product with the extruder, and places it on top of the currently displayed full-size screen. This screen is discussed in detail in See Section 3.14, Print Data Sheet Screen..
  9. “Horn Silence” Button - Clicking this button (or alternately pressing the ctrl +'H' key combination on the keyboard from any APM screen) silences the APM alarm horn. This is useful whenever there is one or more nuisance alarms, the conditions for which keep cycling between true and false, causing the alarm horn to repeatedly begin annunciating shortly after active alarms are acknowledged.
  10. “Restart Data Log Models” Button - This button is disabled for users logged in as “Operator” APM software user types. Clicking this button restarts all data log models. Occasionally, the APM software stops collecting and logging trend data, for reasons that are yet unknown. Restarting the data log models functions as a type of reset that restarts the logging of trend data.
  11. “Command Line” Button - This button is disabled for users logged in as “Operator” APM software user types. Clicking this button brings up a faceplate with which troubleshooting/diagnostic commands can be issued to the APM software.
  12. “Diagnostic Viewer” Button - Clicking this button brings forth the APM Diagnostic Viewer faceplate and places it on top of the currently displayed full-size screen. This screen is available only to users using the Wenger user account, and so it won't be discussed in detail in this document.
  13. “Log Off” Button - Clicking this button causes the current user to be logged out and then logged back in as an “Operator” user type.
  14. “Change Password” Button - Clicking this button brings forth the change password faceplate, shown in Figure 3.31, and places it on top of the currently displayed full-size screen. By using this faceplate the user can change the password for any of the currently defined user types, although to do so he must know the user type's current password to change it to a new one.
  1. “User Name” Text Box - The four user names recognized by the APM application are “Operator”, “Maintenance”, “Supervisor” and “Wenger”.
  2. “Password” Text Box - The “Operator” user name has no associated password while the other three user name do.
  3. “OK” Button
  4. “Cancel” Button
  1. User Name Indicator - The four user names recognized by the APM application are “Operator”, “Maintenance”, “Supervisor” and “Wenger”.
  2. “Old Password” Text Box - This is where the current password is to be entered. The “Operator” user name has no associated password while the other three user name do.
  3. “New Password” Text Box - This is where the new password is to be entered. The “Operator” user name has no associated password while the other three user name do.
  4. “Confirm Password” Text Box - This is where the new password is to be entered again to confirm that no typing errors have been made.
  5. “OK” Button
  6. “Cancel” Button
