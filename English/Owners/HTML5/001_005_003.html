5  Alarms

5.3 Immediate Shutdown Alarms

When any one of the alarms occur that are listed in Table 5.1, the PLC executes an immediate extruder system shutdown (similar to what happens when the APM console E-Stop button is pressed). Each alarm that is listed below is assigned a number for reference only (it does not reflect any priority, severity, or importance). For each immediate shutdown alarm you will find listed below its unique APM alarm reference number and alarm text, both appearing in the alarm region of the screen header and on the Alarm Summary screen. You will also find in Table 5.1, for each alarm, its most probable cause and some troubleshooting suggestions.

Table 5.1 Immediate Extruder System Shutdown
Alarm Number Alarm Text Most Likely Cause Possible Solutions
16 Shutdown:
Low System Air Pressure 		A shutdown has occurred because of a low system air pressure at the facility. See alarm number 37
25 Shutdown:
High/High Extruder Motor Load 		A shutdown has occurred because of a high/high extruder motor load condition. See alarm number 150
27 Shutdown:
Extruder Motor Fault 		A shutdown has occurred because of an extruder motor fault condition. See alarm number 148
