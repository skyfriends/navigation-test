5  Alarms

5.5 Normal Shutdown Alarms

When any one of the alarms occurs that are listed in Table 5.3, the PLC executes a normal extruder system shutdown sequence. Each alarm that is listed below is assigned a number for reference only (it does not reflect any priority, severity, or importance). For each normal shutdown alarm you will find listed below its unique APM alarm reference number and alarm text, both appearing in the alarm region of the screen header and on the Alarm Summary screen. You will also find the most probable cause and some troubleshooting suggestions for each alarm on Table 5.3.
