3.5 Extruder Liquids 1 Screen

3.5.1 Section Overview

The Extruder Liquids 1 screen allows the user to monitor and control the flow rates of the following liquids:

  • Process water to the preconditioner.
  • Process steam to the preconditioner.
  • Process water (different than temperature control water) to the extruder.
  • Process steam (different than temperature control steam) to the extruder.

This screen also displays the most important formula-specific operating conditions on the right-side of the screen from which the corresponding setpoints can also be changed.

The Extruder Liquids 1 screen is accessed by clicking the (F4) Liquid Control button in the region of the Main screen with the forest green background color. It can also be accessed from any screen by pressing the F4 function key on the APM console keyboard.

3.5.2 Details

An example of the Extruder Liquids 1 screen is reproduced in Figure 3.61. This screen shows a PID control loop faceplate for each of the four liquids that are standard for every Wenger extruder system.

  1. Preconditioner Process Water Flow PID Control Loop Faceplate - This faceplate provides the operator with the most options concerning monitoring and changing the control loop's behavior.
  2. Preconditioner Process Steam Flow PID Control Loop Faceplate - This faceplate provides the operator with the most options concerning monitoring and changing the control loop's behavior.
  3. Extruder Process Water Flow PID Control Loop Faceplate - This faceplate provides the operator with the most options concerning monitoring and changing the control loop's behavior.
  4. Extruder Process Steam Flow PID Control Loop Faceplate - This faceplate provides the operator with the most options concerning monitoring and changing the control loop's behavior.
  5. Setpoints and Actual Values of Most Important Operating Conditions - This panel displays the setpoints and current values of the operating conditions that are most crucial to the current product's formulation.
  6. Navigating Buttons - Some screens, such as this one, provide buttons at the bottom of the screen that allow the user to quickly switch to other screens without needing to go to the Main screen first.

The top-right corner of the Extruder Liquids 1 screen has been reproduced in Figure 3.62. The process water/steam flow rates are displayed in the right-most column. To the left is a column that displays, in white background colors, the flow rate setpoints as percentages of the dry feed rate setpoint. To change a water or steam flow rate setpoint, click on the corresponding setpoint in a white rectangle. A faceplate will appear from which you can enter a new setpoint as a percentage of feed rate.

The PID control loop faceplate’s for the four liquids have identical functionalities. Each faceplate allows the operator to place the corresponding flow rate PID control loop in either Auto control mode or Manual control mode. In Auto mode, the PLC varies the respective flow control valve position to maintain a constant flow rate that is determined from the setpoint. In Manual control mode, the PLC maintains the control valve position as per the operator's input. In Figure 3.63, side [A], the extruder steam flow rate control loop is in Auto control mode, while in Figure 3.63, side [B] the extruder steam flow rate control loop is in Manual control mode. The PLC ensures that the flow control valve output remains constant when the operator changes between Auto control mode and Manual control mode; this is referred to as “bumpless” control mode switching.

  1. “Auto” Button - Clicking this button will cause the PLC to switch from Manual control mode to Auto control mode.
  2. “Manual” Button - Clicking this button will cause the PLC to switch from Auto control mode to Manual control mode.
  3. Auto Control Mode Flow Rate Setpoint Select - Indicates the control loop's current control mode, Auto in Figure 3.63, side [A] and Manual in Figure 3.63, side [B].
  4. Auto Control Mode Flow Rate Setpoint Select - This group of controls allow the operator to change the current flow control loop's setpoint, ramping up or down by clicking the arrow buttons, or entering a numerical value by clicking inside the white rectangular field. Notice in Figure 3.63, side [B] that the setpoint is only displayed but cannot be changed when in Manual control mode.
  5. Manual Control Mode Control Valve Position Select - This group of controls allow the operator to change the flow control valve's output command, ramping up or down by clicking the arrow buttons, or entering a numerical value by clicking inside the white rectangular field. Notice in Figure 3.63, side [A] that the flow control valve output command is only displayed but cannot be changed when in Auto control mode.
  6. Setpoint Indicator - Bar graph that displays the current flow rate setpoint in units of weight/hour (converted from percent of dry feed rate). The color of this bar graph, which is always green, doesn't signify anything.
  7. Flow Rate Indicator - Bar graph that displays the current flow rate actual value in units of weight/hour (converted from percent of dry feed rate). The color of this bar graph can change, depending on whether the flow rate is within normal operating range (blue), mildly outside of normal operating range (yellow), or severely outside of normal operating range (red). Notice in Figure 3.63, side [A] that the actual flow rate is with the normal range (blue), but in Figure 3.63, side [B] the actual flow rate is in the mildly excessive range (yellow).
  8. Normal and Alarm Operating Ranges Indicator - This bar graph indicates the three flow rate operating ranges: normal (blue), mildly excessive (yellow), and severely excessive (red).
