3  APM Screens

3.15 Display Languages switching

3.15.1 Section Overview

Where the APM is supplied to Non-English speaking CE countries, Wenger will have all APM screens display specific local language. The user can, at any time, switch back and forth between having the APM text displayed in the local language and having text displayed in English.

3.15.2 Details

In Figure 3.105 the APM software is displaying text in the local language, Spanish in this case.

  1. Language Switching Faceplate - The user has pressed the key combination control+L to bring forth the language switching faceplate by which the user can switch the display language between English and the local language.
  2. “Change to English” Button - Clicking this button causes the display language to be English.
  3. “Change to Local” Button - Clicking this button causes the display language to be in the local language.

In Figure 3.106 the APM display language has been switched to English.
