4 Programmable Logic Controller

4.1 Section Overview

This chapter describes in detail a handful of Wenger-developed Programmable Logic Controller (PLC)-managed, automated extruder system operational sequences that most, if not all customers will frequently use. These startup, shutdown and utility programs are included as part of the PLC code. Customers can do these things themselves manually, of course, but relying on these tried and true Wenger developed automated sequences makes operating the extruder system much easier and less prone to operator error.

The definitions of some terms that are used when the sequences are described are presented first. Then the following extruder system operations and sequences are described in detail:

  • Feeder speed control
  • Live bin level control
  • Normal startup
  • Normal operation after startup
  • Normal shutdown
  • Quick shutdown
  • Dry feed test
  • Preconditioner flush
  • Extruder flush
