4.7 Extruder System Empty Bins Shutdown Sequence

To initiate an PLC-controlled extruder system empty bins shutdown, the operator brings up the Overview screen on the control room APM console, clicks the (F5) Extruder Commands button, and on the Commands Menu dialog box that appears, clicks the Empty Bins Shutdown button. The sequence of operations that the PLC executes when the user chooses an empty bins shutdown are the same as those for a normal shutdown except that the PLC does not begin the shutdown sequence until both the surge bin and the live bin are completely empty of dry feed material.
